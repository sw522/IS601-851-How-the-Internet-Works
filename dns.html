DNS

Pictured Above: Things to consider when choosing a domain name for your website

     The Domain Name System (DNS), is a user-friendly layer to allow devices to look up ip addresses. People are not good at memorizing long strings of numbers so navigating these networks would be very difficult. In order to solve this problem, the DNS was invented to associate these numerical addresses with names that users could remember. When a user enters a domain name to a device it will go query a DNS server in order to look up the proper address. Since there are billions, and soon to be trillions, of possible addresses to store, one server could never be responsible for the entire internet. The solution to this challenge was to set up many DNS server in a distributed hierarchy and are divided into zones which split up the responsibility for the different domains like com, org, net, etc.

     The DNS system was designed to be an open and public communication protocol which does make it susceptible to cyber-attacks. A popular cyber attack is called spoofing, which is when a hacker breaks into a DNS server and changes a particular domain name to be associated with a different IP address of their choosing. The user will be tricked into interacting with an imposter website without ever knowing they are actually sending their information to these criminals. This is one of many reasons proper security and governing bodies are needed to keep billions of people safe while they use the web.
